# 📌 동물 병원 진료 관리 시스템 만들기

<br>

<br>
<br>
<br>

```java

문제: 동물 병원 진료 관리 시스템 만들기
- 동물 병원에서 사용하는 진료 관리 시스템을 만들어봅시다.
  동물의 기본 정보를 관리하는 Animal 추상 클래스를 만들고,
  이를 상속받는 구체적인 동물 클래스들과 진료 기록을 관리하는
  클래스를 구현하세요.


요구사항:
1. Animal 추상 클래스를 만드세요.
2. 클래스는 동물의 기본 정보를 저장할 수 있는 필드들을 가집니다.
3. 다음 기능들을 구현하세요:


기본 필드:
- id (등록번호)
- name (이름)
- age (나이)
- ownerName (보호자 이름)
- ownerContact (보호자 연락처)


필수 메서드:
- 추상 메서드 getSpecialCare(): 동물별 필수 관리사항 반환
- 일반 메서드 displayInfo(): 동물 기본 정보 출력


상속 클래스 구현:
1. Dog 클래스:
   - 추가 필드: breed(견종), isNeutered(중성화여부)
   - 추상 메서드 getSpecialCare()
   // 출력 예시
   // [강아지 관리 사항]
    // 1. 매일 2회 산책이 필요합니다.
    // 2. 심장사상충 예방약 매월 복용이 필요합니다.

2. Cat 클래스:
   - 추가 필드: isIndoor(실내사육여부)
   // 출력 예시
  //  [고양이 관리 사항]
  //   1. 매일 화장실 청소가 필요합니다.
  //   2. 실내 스크래치 포스트를 제공해야 합니다.

3. Bird 클래스:
   - 추가 필드: species(종), canSpeak(말하기가능여부)
   // 출력 예시
  //  [새 관리 사항]
    // 1. 하루 2번 이상 새장 청소가 필요합니다.
    // 2. 햇빛을 충분히 쬐어줘야 합니다.


진료 기록 클래스 구현:
- MedicalRecord 클래스 생성
- 필드: recordId(진료번호), animalId(환자번호), visitDate(진료일), symptoms(증상), diagnosis(진단), treatmentCost(진료비)
- 메서드: displayTreatmentInfo() - 진료 정보 출력



// 전체 시스템 출력 예시
Dog dog = new Dog("D001", "멍멍이", 3, "홍길동", "010-1234-5678", "골든리트리버", true);
dog.displayInfo();
// === 강아지 정보 ===
// 등록번호: D001
// 이름: 멍멍이
// 나이: 3세
// 보호자: 홍길동
// 연락처: 010-1234-5678
// 품종: 골든리트리버
// 중성화 여부: 완료

// [강아지 관리 사항]
// 1. 매일 2회 산책이 필요합니다.
// 2. 심장사상충 예방약 매월 복용이 필요합니다.

MedicalRecord record = new MedicalRecord("R001", "D001", LocalDate.now(),
    "기침과 콧물", "감기", 50000);
record.displayTreatmentInfo();
// === 진료 기록 ===
// 진료 번호: R001
// 환자 번호: D001
// 진료 일자: 2025-02-02
// 증상: 기침과 콧물
// 진단: 감기
// 진료 비용: 50,000원




Cat cat = new Cat("C001", "나비", 2, "김영희", "010-9876-5432", true);
cat.displayInfo();
// === 고양이 정보 ===
// 등록번호: C001
// 이름: 나비
// 나이: 2세
// 보호자: 김영희
// 연락처: 010-9876-5432
// 실내사육 여부: 예

// [고양이 관리 사항]
// 1. 매일 화장실 청소가 필요합니다.
// 2. 실내 스크래치 포스트를 제공해야 합니다.


Bird bird = new Bird("B001", "짹짹이", 1, "김철수", "010-5555-5555", "앵무새", true);
bird.displayInfo();
// === 새 정보 ===
// 등록번호: B001
// 이름: 짹짹이
// 나이: 1세
// 보호자: 박철수
// 연락처: 010-5555-5555
// 종: 앵무새
// 말하기 가능 여부: 가능

// [새 관리 사항]
// 1. 하루 2번 이상 새장 청소가 필요합니다.
// 2. 햇빛을 충분히 쬐어줘야 합니다.



```
